# Try here #
https://bjornt18.github.io/ReactSortingVisualizer

I created this app to help myself and others better understand how sorting algorithms work.

# Main features #
* Sorting algorithms
  * Selection sort
  * Insertion sort
  * Bubble sort
  * Quick sort
  * Merge sort
* Speed slider
* Selection controls
  * Skip to beginning
  * One step backwards
  * Play/Pause
  * One step forwards
  * Skip to end

# Run locally #
Install dependencies, in the project directory, run:

### `npm install`

To start, in the project directory, run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
